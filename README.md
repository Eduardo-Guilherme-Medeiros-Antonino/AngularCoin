# AngularCoin 🪙

## 📝 Descrição do Projeto
[cite_start]O **AngularCoin** é um aplicativo web desenvolvido para a atividade final complementar[cite: 1]. [cite_start]Trata-se de um app de conversão de moedas em tempo real que consome dados de APIs REST externas para permitir que os usuários consultem taxas de câmbio atualizadas e realizem conversões entre diferentes moedas globais de forma simples e intuitiva[cite: 3, 6].

---

## 🚀 Funcionalidades
- [cite_start]**Conversão em Tempo Real:** Integração com API REST para taxas de câmbio atualizadas[cite: 19, 21].
- [cite_start]**Histórico de Conversões:** Armazenamento local das consultas recentes (via Local Storage)[cite: 28].
- [cite_start]**Modo Offline:** Consulta às últimas taxas armazenadas caso não haja conexão com a internet[cite: 37].
- [cite_start]**Interface Responsiva:** Design adaptável para dispositivos móveis e desktop[cite: 23, 26].
- [cite_start]**Conversão Inversa:** Inversão rápida entre moeda de origem e destino com um clique[cite: 34].

---

## 🛠️ Tecnologias Utilizadas
- [cite_start]**Framework:** Angular [cite: 44]
- [cite_start]**Linguagem:** TypeScript [cite: 45]
- [cite_start]**API:** ExchangeRate-API / CurrencyConverterAPI [cite: 46, 47]
- [cite_start]**Estilização:** HTML5 / CSS3 (Interface Responsiva) [cite: 24]
